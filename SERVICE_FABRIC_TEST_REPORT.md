## 1. System Insights & Findings

### 1.1 Core Flask Service (`core_flask_service`)
- **Status:** Running.
- **Service Mounting:** Successfully mounts Blueprints from `6_service_catalog`.
- **Issues:**
    - **Template Error:** `jinja2.exceptions.UndefinedError: 'service' is undefined` when using `smart_render`. This happens because `fallback_wrapper.html` (which extends `service_wrapper.html`) expects a `service` object in the context to display the ribbon, but `smart_render` in `app/utils.py` doesn't provide it.
    - **Service Failures:** 
        - `file-formats-mermaid` fails to mount due to missing `GEMINI_API_KEY`.
        - `gemini_agent` fails to mount due to missing `GEMINI_API_KEY`.
- **Observation:** Mounting 45 services takes significant time, justifying the previous timeout increase.

### 1.2 Central Gateway (`fastapi_core`)
- **Status:** 🔴 CRASHING / 502 Bad Gateway.
- **Issue:** `ModuleNotFoundError: No module named 'chromadb'`. The vector store service depends on `chromadb`, which is missing from the container's environment.
- **Impact:** Real-time events (WebSockets) and Vector Store operations are currently unavailable.

### 1.3 Backend API (`backend_api`)
- **Status:** 🟢 Running.
- **Database:** Migrations are applied. Templates and Instances are correctly seeded/present in the database.
- **Authentication:** JWT secret is shared with Flask Core.

### 1.4 Proxy (`proxy`)
- **Status:** 🟢 Running.
- **Routing:** Correctly forwards `/` to the static frontend and `/app/core/` to the Flask service.
