# Transition from Wave 8

Wave-9 consumes the frozen Wave-7 agentic contracts and completed Wave-8 provider execution surface without reopening either. The Wave-8 integration record marks all provider-runtime, adapter, LangGraph, and evaluation lanes integrated and complete.

Provider execution remains owned by `ProviderExecutionService`. Wave-9 submits only canonical work derived from `AgentRunPlan`, consumes normalized results and evidence, and persists only factory-specific approvals, review decisions, unmet requirements, and handoff references.
