# Integration order

1. Freeze shared factory contracts and write the contract-first journey.
2. Integrate reviewed technology-profile and blueprint-compiler outputs.
3. Integrate factory lifecycle and repository bootstrap services.
4. Integrate candidate review and application integration services.
5. Compose the factory service and CLI, install locked runtime dependencies, and run the Wave-9 verification target.
6. Record integration decisions and closure; do not merge into `main` automatically.

Every candidate starts from the exact approved integration base. Only accepted exact commit SHAs may be applied to the integration branch. Provider dispatch, recovery, policy enforcement, and result normalization remain Wave-8 responsibilities.
